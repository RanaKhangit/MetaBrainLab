## Implementation Notes

1. **Favicon:** Convert `favicon.svg` to ICO format with sizes 16x16, 32x32, 48x48
2. **OG Image:** Convert `og-image-metabrain.svg` to PNG at 1200x630px
3. **Animations:** The animated SVGs include CSS keyframes — ensure they render correctly
4. **Colours:** All logos use the brand colours specified in the Developer Brief
